# Renommage de fichiers

Ce dossier contient des scripts pour renommer automatiquement des fichiers avec un préfixe et une numérotation.

- `renomme_dossier_unique.py` : script de renommage d’un dossier spécifique (voir `fiche1.md` pour le mémo détaillé).
- `renomme_argparse.py` : script générique où l’on passe le dossier et le préfixe en arguments (voir `fiche2.md`).

Les fichiers `fiche1.md` et `fiche2.md` expliquent le fonctionnement ligne par ligne et donnent des exemples d’utilisation.
