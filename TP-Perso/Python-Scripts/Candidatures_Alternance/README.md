# Candidatures alternance automatisées

Ce dossier contient un script pour envoyer automatiquement le même mail de candidature à une liste d’entreprises.

- `candidatures_ais.py` : script d’envoi d’emails avec pièces jointes (voir `fiche4.md` pour le mémo détaillé).

Le fichier `fiche4.md` décrit le fonctionnement du script, les fichiers nécessaires et les erreurs courantes.
