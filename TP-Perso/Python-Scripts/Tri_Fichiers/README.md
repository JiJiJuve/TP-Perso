# Tri de fichiers par extension

Ce dossier contient des scripts pour ranger un dossier en créant des sous-dossiers par extension (PDF, JPG, DOCX, etc.).

- `tri_fichiers.py` : script de tri automatique d’un dossier (voir `fiche3.md` pour le mémo détaillé).
- `tri_fichiers_dry_run.py` : même script avec une option `--dry-run` pour simuler le tri sans déplacer les fichiers (voir `fiche6.md`).

Les fichiers `fiche3.md` et `fiche6.md` expliquent le fonctionnement ligne par ligne et donnent des exemples d’utilisation.
